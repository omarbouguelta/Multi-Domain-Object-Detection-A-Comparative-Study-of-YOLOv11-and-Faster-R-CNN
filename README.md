# Multi-Domain-Object-Detection-A-Comparative-Study-of-YOLOv11-and-Faster-R-CNN
This repository contains the implementation and comparative analysis of state-of-the-art object detection architectures—YOLOv11 (One-Stage) and Faster R-CNN (Two-Stage). The project evaluates performance across three distinct environmental domains: Public Safety, Traffic Infrastructure, and Remote Sensing.
